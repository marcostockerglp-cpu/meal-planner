# Smart Meal Planner – Web App MVP v8

## Start lokal

```bash
npm install
npm run dev
```

## Neu in dieser Version

- Die Detailansicht der Rezepte ist jetzt scrollbar
- Lange Rezepte und viele Schritte können vollständig nach unten gelesen werden
- Der Header der Kochansicht bleibt sichtbar
- Neuer optionaler Modus: **Step-by-Step**
- Standard bleibt die bisherige komplette Rezeptansicht
- Step-by-Step kann manuell in der Kochansicht aktiviert werden

- Header der Detailansicht ist nicht mehr sticky
